    5. 目前这套博客的发布流程

    当前博客的实际流程就是：

    1. 在根目录写好 Markdown 文章。
    2. 把文章放到 source/_posts/。
    3. 补上标题、时间和分类。
    4. 执行 npm.cmd run build。
    5. 执行 npm.cmd run deploy。

    后续如果你继续写新文章，按这个流程处理就可以了。
